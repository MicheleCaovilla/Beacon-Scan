This an Android application that uses Bluetooth beacons, in particular easiBeacon.

The aim is to calculate the of position of the mobile phone where is executed the app, using the distances from the beacons and the Nelder-Mead Simplex Method.

iOS application can be found here, developed from my friend @Tomfox91.

The app was created and tested on my Google Nexus 4.
